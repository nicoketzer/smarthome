Zusätzliche Infos:

zu *1: Sollten keine feste IP-Adresse vom ISP zugeteilt werden so ist zwingend eine DDNS-Adresse nötig! Sollten Sie keine DDNS-Adresse haben können sie diese gerne bei german-backup.de einrichten. Es werden aber auch DDNS-Adressen anderer Anbieter akzeptiert.
zu *2: Hierbei ist der HTTP-Port gemeint. Da die Kommunikation zw. Webinterface und C&C-Server sowieso nur über verschlüsselte Token stattfindet. Sollte der HTTP-Port dem Standart-Port entsprechen so ist das Feld frei zu lassen oder 80 einzutragen
zu *3: Hierbei handelt es sich um den Hostnamen den der C&C-Server in ihrem Internen Netzwerk von ihrem Router zugeteilt bekommen hat. Diesen können sie z.B. bei einer Fritz-Box unter dem Punkt "Heimnetzwerk"->"Mesh" oder "Heimnetzwerk"->"Netzwerk" herausfinden.
zu *4: Hiermit ist der Name gemeint unter welchen der Server in der Web-Oberfläche für Admin´s sichtbar ist.
